meta属性详解

meta标签是HTML中一个很重要的标签，它定义在每个html文档中的head区，许多初学者在学习的过程中可能不会先接触到，因为各种学习资料中更多的去介绍了html中的内容标签，而meta标签的内容并不会在网页上直接显示，所以导致很多人忽视了这个标签。
但如果去看那些大型网站的源码，可以发现有着许多meta标签，并且这些标签都有着很重要的意义，用好meta标签，会带来很多意想不到的效果。一个好的meta标签可以大大提高网站被搜索到的可能，还可以用其来设定页面格式以及刷新等功能，因此我们有必要对meta标签的各种属性进行了解，并在网站上灵活运用。

meta标签的属性分类

meta标签的属性分为两类，http-equiv属性以及name属性。

http-equiv属性

顾名思义，http-equiv属性相当于http的头文件，它向浏览器传回了一些有用的信息，以此来使浏览器准确的显示网页内容，他的属性值为content，它的定义格式为：
<meta http-equiv="参数" content="参数变量值">

这其中常用的参数分为以下几种：

Expires

Expires属性的作用是设置该网页缓存的过期时间，如果网页过期，那么与服务器重新传输，该属性能够设定两个值：

  • GMT时间，如
    <meta http-equiv="Expires" Content="Fri, 1 Jan 2016 12:11:11 GMT">
  • 0，数字表示多长时间后过期，如
    <meta http-equiv="Expires" content="0">

Content-Type

Content-Type属性用来设定页面使用的字符集，如gb2312，utf-8等，如
<meta http-equiv="Content-Type" content="text/html;charset=utf-8">

Content-Language

Content-Language属性用来设定网页所使用的语言，如
<meta http-equiv="Content-Language" content="zh-CN">

Date

Date属性用来制定页面创建的时间，content使用GMT时间格式，如
<meta http-equiv="Date" content="Fri, 1 Jan 2016 12:11:11 GMT">

Cache-Control

Cache-Control属性非常重要，用来控制网页的缓存机制，其值通常有以下几种

  • public：所有内容都将被缓存
  • private：内容只缓存到私有缓存中
  • no-cache：所有内容都不会被缓存
  • no-store：所有内容都不会被缓存到缓存或Internet临时文件中

Pragma

Pragma属性是Cache-Control属性的旧版本，Pragma属性是HTTP/1.0的接口，而Cache-Control是HTTP/1.1的接口

Last-Modified

Last-Modified属性定义网页最后一次修改的时间，属性值使用GMT时间格式，如
<meta http-equiv="Last-Modified" content="Fri, 1 Jan 2016 12:11:11 GMT">

Refresh

Refresh属性定义网页自动刷新时间并能够指向新的网页，如
<meta http-equiv="Refresh" content="2; URL=kipbase.github.io">
网页会在两秒后自动刷新到指定URL地址，如果content只写入了一个数字，那么就是在n秒后刷新页面

Set-Cookie

设定一个Cookie，设定格式与JS相同，如
<meta http-equiv="set-cookie" content="name=kipbase;expires=Fri, 30 Dec 2011 12:00:00 GMT; path=/">

Window-Target

设定显示窗口，强制页面在当前窗口以独立页面的形式显示，用于防止其他人在框架中调用自己的页面，如
<meta http-equiv="Window-target" content= "_top">

name属性

name属性用来当定义HTML文档的描述，如关键词，作者等等，其定义格式为：
<meta name="参数" content="参数变量值">
其主要属性为以下几种

author

author属性用来定义作者的名字，如
<meta name="author" content="Kipbase">

description

description属性定义页面的描述，搜索引擎会将这个描述显示在搜索结果中，如
<meta name="description" content="Kipbase Personal Blog">

keywords

keywords属性用来定义关键词，这些关键词有主意搜索引擎对页面进行分类，如
<meta name="keywords" content="HTML, CSS, JavaScript">

robots

robots属性用来告诉搜索机器人哪些页面需要索引，哪些不需要，可选的值为all,none,index,noindex,follow,nofollow。默认为all，如
<meta name="robots" content="none">

其它属性

相比于基本的name和http-equiv属性，还有一些定义在特定情况下的属性，如控制缩放，控制兼容等等。

meta标签的作用

meta标签的作用主要是对网页进行一些控制，如缓存，显示窗口等等，还有就是对搜索引擎的控制，其主要作用分为以下几点。

  • 帮助各种搜索引擎进行准确的索引，合理的设置关键词描述等能够提高准确性，并且提高访问量。
  • 简单地实现自动刷新功能，不用使用复杂的JS。
  • 控制页面的缓存和Cookie。
  • 控制页面的显示窗口，防止框架调用。
  • 一些高级功能，如页面评级，转换动画，这些功能在这篇内容中不做赘述，有需要的可去网上进行查询。
